Cron Syntax

This is a quick reference to cron syntax and also shows the options supported by node-cron, the library used to task schaduling.

Allowed fields

{% highlight text %} ┌────────────── second (optional) │ ┌──────────── minute │ │ ┌────────── hour │ │ │ ┌──────── day of month │ │ │ │ ┌────── month │ │ │ │ │ ┌──── day of week │ │ │ │ │ │ │ │ │ │ │ │ * * * * * * {% endhighlight %}

Allowed values

Field Value
second 0-59
minute 0-59
hour 0-23
day of month 1-31
month 1-12 (or names)
day of week 0-7 (or names, 0 or 7 are sunday)

Using multiples values

Runnin a task every second 1, 2, 10 and 14. {% highlight json %} { "tasks": [ { "name": "My task", "path": "./path/to/my_task.js", "cron": "1,2,10,14 * * * * *" } ] } {% endhighlight %}

Using ranges

Runnin a task from second 1 to 10. {% highlight json %} { "tasks": [ { "name": "My task", "path": "./path/to/my_task.js", "cron": "1-10 * * * * *" } ] } {% endhighlight %}

Using step values

Runnin every 2 seconds {% highlight json %} { "tasks": [ { "name": "My task", "path": "./path/to/my_task.js", "cron": "*/2 * * * * *" } ] } {% endhighlight %}

Using names

Scheduling a task to be performed every Sunday in January at 00:00. {% highlight json %} { "tasks": [ { "name": "My task", "path": "./path/to/my_task.js", "cron": "0 0 0 * January Sunday" } ] } {% endhighlight %} Or using short names. {% highlight json %} { "tasks": [ { "name": "My task", "path": "./path/to/my_task.js", "cron": "0 0 0 1 Jan Sun" } ] } {% endhighlight %}
