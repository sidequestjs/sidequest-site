Getting Started

Four steps to multi-process tasks:

1 - Install the sidequest dependency:

{% highlight text %} npm install --save sidequest {% endhighlight %}

2 - Create a task file:

{% highlight javascript %} exports.run = () => { console.log('Hello!'); } {% endhighlight %}

3 - Register you task at sidequest-config.json:

{% highlight json %} { "tasks": [ { "name": "Hello task every 2 seconds", "path": "./path/to/hello.js", "cron": "*/2 * * * * *" }, { "name": "Hello task every 5 seconds", "path": "./path/to/hello.js", "cron": "*/5 * * * * *" }, ] } {% endhighlight %}

4 - Initialize sidequest in you app:

{% highlight javascript %} const sidequest = require('sidequest'); // ... sidequest.initialize(); // ... {% endhighlight %}
